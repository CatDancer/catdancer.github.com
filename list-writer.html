list-writer

A writer for Arc lists.

arc2 writes Arc lists by converting them to Scheme lists and printing them using the Scheme writer. By implementing a writer for Arc values we can build upon this patch to customize the output of Arc values. The table-reader-writer patch does this for Arc tables.

By itself this patch is supposed to produce the same output as arc2 does. (If it doesn’t, that would be a bug).

Bugs

  • Attempting to write a list containing a cycle will enter an infinite loop and crash MzScheme by consuming all available memory.
Have fixes? Send them to me on GitHub.

Apply This Hack to Your Arc

By using git or patch, you can incorporate this patch into your version of Arc.

With git

 $ git pull git://github.com/CatDancer/arc.git tag arc2.list-writer0

For example,

 $ mkdir arc
 $ cd arc
 $ git init
 $ git pull git://github.com/CatDancer/arc.git tag arc2.list-writer0

With patch

To apply this patch to your copy of arc using the patch command, download arc2.list-writer0.patch into your Arc directory, and at the Unix command line inside the Arc directory, type:

 patch <arc2.list-writer0.patch

“patch” is a standard utility that comes with Unix. (If you’re running Windows, iirc patch comes with cygwin).

For example,

 $ wget http://ycombinator.com/arc/arc2.tar
 [... downloading ...]
 $ tar xf arc2.tar
 $ cd arc2
 $ patch <arc2.list-writer0.patch
 patching [...]
 $
