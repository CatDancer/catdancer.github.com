Arc hacks:

exit-on-eof

Patches Arc so that an eof (^D in Unix) at the command prompt will exit Arc.

$ mzscheme -m -f as.scm
Use (quit) to quit, (tl) to return here after an interrupt.
arc> ^D
$

Get This Hack

  • diff against arc2
  • repository on GitHub
  • download a tarball of arc2 with this patch applied

Applying This Hack to Your Arc

By using patch or git, you can incorporate this patch into your version of Arc.

With patch

To apply this patch to your copy of arc using the patch command, download exit-on-eof.patch into your Arc directory, and type:

patch <exit-on-eof.patch

With git

To apply this patch using git, start with a git repository containing arc2 or the version of Arc that you want to patch. Then in the git working directory, type:

git pull git://github.com/CatDancer/arc-exit-on-eof.git master
