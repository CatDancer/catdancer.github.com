exit-on-eof

Patches Arc so that an eof (^D in Unix) at the command prompt will exit Arc.

$ mzscheme -m -f as.scm
Use (quit) to quit, (tl) to return here after an interrupt.
arc> ^D
$

Get This Hack

  • diff against arc2
  • repository on GitHub
  • download a tarball of arc2 with this patch applied

Apply This Hack to Your Arc

By using git or patch, you can incorporate this patch into your version of Arc.

With git

 $ git pull git://github.com/CatDancer/arc.git tag arc2.exit-on-eof0

For example,

 $ mkdir arc
 $ cd arc
 $ git init
 $ git pull git://github.com/CatDancer/arc.git tag arc2.exit-on-eof0

With patch

To apply this patch to your copy of arc using the patch command, download arc2.exit-on-eof0.patch into your Arc directory, and at the Unix command line inside the Arc directory, type:

 patch <arc2.exit-on-eof0.patch

“patch” is a standard utility that comes with Unix. (If you’re running Windows, iirc patch comes with cygwin).

For example,

 $ wget http://ycombinator.com/arc/arc2.tar
 [... downloading ...]
 $ tar xf arc2.tar
 $ cd arc2
 $ patch <arc2.exit-on-eof0.patch
 patching [...]
 $
