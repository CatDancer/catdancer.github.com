Arc hacks:

json

alpha release of a JSON parser / generator

JSON is a lightweight data-interchange format. It is a subset of JavaScript, and so is especially popular for web applications.

 arc> (prn:tojson `(1 "a" ,(obj x 'null y 'true)))
 [1,"a",{"y":true,"x":null}]

 arc> (fromjson "[true,false,null,[],42]")
 (t nil nil nil "42")

Bugs

  • fromjson returns numbers as an unparsed string.
  • Whitespace is not parsed.
  • fromjson does not distinguish between false, null, and the empty array, if that’s something you need for your application.
  • tojson does not recognize non-integer numbers.
  • Lots of duplicate and ugly code in the implementation.
Have fixes? Send them to me on GitHub.
